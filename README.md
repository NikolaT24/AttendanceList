## ✨ Overview

This project provides a **fully automated LaTeX template** for generating attendance sheets.

Instead of manually formatting tables, the system:
- Generates rows dynamically  
- Maintains consistent spacing  
- Produces clean, printable documents  

---

## ⚙️ How It Works

At its core, the system uses **LaTeX3 (`expl3`)** to generate table rows programmatically.
